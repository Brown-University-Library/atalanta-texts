Der König / dem sein Haupt ist von einer güldnen Kronen ganz schwer /
Rufft also uberlaut / schwimmende im tiessem Meer:
Warumb helfft ihr mir nicht? Warumb kompt nicht g'elauffen jederman /
So ich errettet auß Nohe deß Wassers / begaben kan /
Bringt mich / so ihr weiß seijt / in mein Reich / und es sol zu keiner Zeit
Euch Armuht trucken / oder beschwerlich Leibes Kranckheit.
